# Wechat_JumpGame
Wechat_JumpGame
自己写的一个微信跳一跳自动游戏程序。

* 绕过最新反作弊验证。

* 全自动运行,失败自动重来
* 自动适应不同分辨率
* 自动调整各个参数误差，玩的越久表现越好~

不需要配置adb,不需要配置python环境

只需要

1. 安卓手机，开启开发者模式，允许开发者安全模式（允许模拟触屏）
2. 手机连接到电脑
3. 双击"运行.exe"

喝茶去吧!

如果一开始效果不好，
请适当调整K_array文件夹下的k0.txt里的数字（全局初始系数），直到第一次跳准[第一次跳的远就适当减小数字，跳得近就适当增加数字】(它可以继续不断自动微调)

--------------------------
源代码在Source_Code文件夹里

第一次运行请先运行require.bat

编译exe请运行buildexe.bat
